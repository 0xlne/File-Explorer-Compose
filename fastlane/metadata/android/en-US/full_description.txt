<p><i>File Explorer</i> is a full-featured and lightweight file managing app written in kotlin with Jetpack Compose UI.</p><p><br><b>Features:</b></p><ul><li>Free, lightweight and open source. No ads, subscriptions or in-app purchases.</li><li>Simple and modern declarative UI design using Jetpack Compose.</li><li>All basic file management functionality (e.g. cut, copy, delete, compress, extract etc) are supported.</li><li>Lets you open multiple tabs to make file managing a lot easier.</li><li>built-in media viewers (Images, Videos, Audios and PDFs are supported).</li></ul>
